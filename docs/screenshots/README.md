# Screenshot examples

The project README documents the five supplied interface examples: English profile, nearby search, journey overlay, influence lists, and Turkish localization.

The image attachments from the design review are not files in the repository, so copy them here manually before committing:

- `01-profile-en.png`
- `02-nearby-search.png`
- `03-life-journey.png`
- `04-influences.png`
- `05-profile-tr.png`

Capture fresh screenshots with `npm run dev` at `http://localhost:3000/en` and `http://localhost:3000/tr`. Keep screenshots free of `.next` cache paths and browser personal data.
